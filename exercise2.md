# GitHub first exercise

### Commands


``` bash
- git init
- git add
- git commit -m 
- git status
- git pull
- git push
```
### git init

- initialize git repo

### git add  filename 

- add file to git repo

### git commit -m

- commit your changes to repo

### git status

- show current git status

### git pull

- pull changes from server

### git push

- push changes to server
